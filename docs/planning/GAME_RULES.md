## 2. 직업 (Role) 명세

### 2.1 시민 진영

#### 🏙️ 시민 (Citizen)
- **능력**: 없음
- **승리 조건**: 모든 마피아 제거
- **설명**: 낮 투표에서 마피아를 추리하는 것이 유일한 무기

#### 🚓 경찰 (Detective)
- **능력 (밤)**: 대상 1명의 진영(마피아 여부)을 확인
- **승리 조건**: 모든 마피아 제거
- **제한**: 같은 대상을 연속 2번 조사 불가
- **설명**: 정보 수집의 핵심. RAG 지식베이스로 수집한 정보를 낮 토론에 활용

#### 🏥 의사 (Doctor)
- **능력 (밤)**: 대상 1명 보호 (마피아 공격 무효화)
- **승리 조건**: 모든 마피아 제거
- **제한**: 연속 2회 같은 대상 보호 불가 / 자기 자신 보호는 게임당 1회만 허용
- **설명**: 마피아 공격을 차단하는 핵심 수비 직업

#### 🔮 점쟁이 (Fortune Teller)
- **능력 (밤)**: 대상 1명의 직업군(시민계열/마피아계열/중립) 확인
- **승리 조건**: 모든 마피아 제거
- **제한**: 경찰보다 넓은 범위이나 정확도가 낮음 (직업군 레벨)
- **설명**: 12인 이상 게임에서 경찰을 보조하는 정보원 역할

---

### 2.2 마피아 진영

#### 🔫 마피아 (Mafia)
- **능력 (밤)**: 대상 1명 지목 (팀 내 의결 후 최종 1명 처치)
- **승리 조건**: 마피아 수 ≥ 생존 시민 수
- **공격 방식**: 마피아가 여러 명일 때 각자 1명씩 지목 → 슈퍼바이저가 최종 1명 결정
- **설명**: 낮에는 시민인 척 행동, 밤에 협의하여 대상 제거

#### 🗡️ 킬러 (Killer)
- **능력 (밤)**: 의사의 보호를 무시하고 처치 가능 (게임당 1회)
- **승리 조건**: 마피아 수 ≥ 생존 시민 수
- **제한**: 관통 공격은 게임당 1회 한정
- **설명**: 의사를 무력화하는 고급 마피아 직업. 15명 이상 시 등장

---

### 2.3 중립 진영

#### 🎭 광대 (Jester)
- **능력**: 없음 (특수 승리 조건만 존재)
- **승리 조건**: 낮 투표에서 처형당하면 **단독 승리**
- **패배 조건**: 마피아에 의해 밤에 죽거나, 게임 종료 시 생존 시 패배
- **설명**: 의심받는 행동을 유도하며 투표에서 처형되는 것이 목표

#### 🕵️ 스파이 (Spy)
- **능력 (밤)**: 마피아 채팅 채널 엿듣기 가능 (읽기 전용)
- **승리 조건**: 게임 종료 시까지 생존하면 승리
- **설명**: 정보를 수집해 생존하는 것이 목표. 15명 이상 시 등장

## 3. 게임 진행 흐름 (Game Flow)

```
[게임 시작]
    │
    ├─→ 직업 배분 (랜덤, 플레이어에게 본인 직업만 공개)
    │
    ▼
[낮 Phase]
    ├─ 1. 전체 채팅 (3분)
    ├─ 2. 투표 Phase (1분) → 최다 득표자 1명 후보
    ├─ 3. 최후의 변론 (30초) → 후보자 마지막 발언
    └─ 4. 최종 찬반 투표 (30초) → 처형 or 무효
    │
    ▼
[밤 Phase]
    ├─ 1. 마피아 협의 채널 오픈 (1분 30초)
    │      → 마피아끼리만 볼 수 있는 채팅
    │      → 슈퍼바이저가 최종 1명 결정
    ├─ 2. 각 직업별 능력 사용 (1분)
    └─ 3. 밤 결과 처리 → 마피아 공격 vs 의사 보호 대조
    │
    ▼
[승리 조건 판정]
    ├─ 광대 처형됨 → 광대 단독 승리
    ├─ 마피아 수 ≥ 생존 시민 수 → 마피아 승리
    ├─ 마피아 전원 제거 → 시민 승리
    └─ 조건 미충족 → 낮 Phase로 반복
```

## 4. 투표 상세 규칙

### 4.1 일반 투표
- 투표 시간: 1분
- 동률 처리: 재투표 1회 실시
- 재투표 후 동률: 해당 라운드 처형 없음 (무투표)
- AI Agent는 반드시 투표 (기권 AI는 랜덤 대상)

### 4.2 최후의 변론 투표
- 처형 대상 생존 여부를 찬반으로 재결정
- 과반수 찬성(처형): 처형 확정
- 과반수 반대(생존): 처형 취소
- 동률: 처형 확정

### 4.3 특이 케이스
- 사망자는 투표, 발언 불가
- 광대가 최다 득표 시 → 광대 승리 선언 및 게임 즉시 종료

---

## 5. 특수 이벤트 룰

### 5.1 첫날 밤 규칙
- 첫날 밤은 마피아 공격 없음 (탐색 밤)
- 경찰/의사 능력 사용은 가능

### 5.2 마피아 전멸 직전 규칙
- 마지막 마피아 1명 남았을 때 `도망` 선언 가능 (1회)
- 도망 선언 시 처형 투표 무효 1회 적용

### 5.3 연속 보호/조사 제한
- 의사: 동일 대상 연속 2회 보호 불가
- 경찰: 동일 대상 연속 2회 조사 불가

### 5.4 침묵 패널티
- 낮 3분 동안 한 번도 발언하지 않은 AI Agent는 `침묵` 상태로 표시
- 침묵 AI는 해당 라운드 투표 시 -1 신뢰도 패널티

## 6. 승리 조건 상세

| 진영 | 승리 조건 | 패배 조건 |
|------|----------|----------|
| 시민 | 모든 마피아 제거 | 마피아 수 ≥ 생존 시민 |
| 마피아 | 마피아 수 ≥ 생존 시민 | 모든 마피아 제거 |
| 광대 (중립) | 낮 투표로 처형됨 | 밤 사망 / 끝까지 생존 |
| 스파이 (중립) | 게임 종료 시 생존 | 사망 |

> 광대 승리는 독립적이므로 광대가 처형되는 즉시 게임 종료 (시민/마피아 승패 무관)

## 7. 정보 공개 규칙

| 이벤트 | 공개 정보 |
|--------|----------|
| 낮 처형 | 사망자 이름 + 직업 (기본 공개) |
| 밤 사망 | 사망자 이름 + 직업 (기본 공개) |
| 경찰 조사 | 경찰 본인에게만 결과 전달 |
| 의사 보호 성공 | 의사 본인에게만 성공 여부 전달 |
| 킬러 관통 | 의사에게 보호 실패 알림 |
| 스파이 엿듣기 | 스파이 본인에게만 마피아 채팅 표시 |

## 8. AI 발언 가이드라인

- AI Agent는 자신의 **직업에 맞는 언어 전략**을 사용
- 마피아 AI: 의심을 다른 사람에게 돌리는 발언, 적극적인 방어
- 경찰 AI: 정보를 선택적으로 공개 (본인 신분 보호하며)
- 광대 AI: 의도적으로 의심받는 발언 유도
- 발언 타이밍은 랜덤 딜레이(1~15초) 적용으로 자연스러움 확보
