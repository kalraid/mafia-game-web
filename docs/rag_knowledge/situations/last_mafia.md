# 마피아 1명 남은 상황 대응

category: situation
phase: endgame
condition: 마피아 최후 1인 생존

---

## 마피아 관점 (1명 생존)

### 절대 원칙
- 절대 서두르지 않는다. 시민들이 판단을 미루게 만들어야 함
- 조용한 중재자 포지션: "저는 어느 편도 아닌데, 두 분 의견 다 이해가 돼요"
- 의심이 다른 사람에게 분산되어 있는 한 생존

### 도망 선언 활용
```
도망 선언 타이밍 (게임당 1회):
- 최후의 변론에서 찬성이 과반일 것이 확실한 순간
- 직전에 도망 선언 → 그 라운드 처형 투표 무효
- 다음 밤 결정적 인물 제거 후 역전 시도
```

### 역전 시나리오
```
현재: 시민 3명 vs 마피아 1명 → 처형 실패 시:
밤 공격 → 시민 2명 vs 마피아 1명 → 마피아 수 ≥ 시민 수 → 마피아 승리!

역전 조건: 처형 1회 무효화 + 밤 공격 성공
```

### 발언 전략
```
피해야 할 발언:
- 강한 주장 (의심 유발)
- 특정 시민을 강하게 지목 (티남)
- 팀원 보호 흔적

해야 할 발언:
- "저도 잘 모르겠어요. 다들 어떻게 생각하세요?"
- "두 사람 중 한 명이겠죠. 저는 Y가 조금 더 이상한 것 같기도 해요" (가벼운 암시)
- "이번에 실수하면 우리가 지니까 신중하게 봐요"
```

## 시민팀 관점 (마피아 1명 남은 상황)

### 조여드는 추리
- 이미 사망자 직업이 공개되어 있음 → 역추적으로 마피아 공격 패턴 분석
- 경찰이 살아있다면 마지막 의심 인물 조사 → 즉시 확인 가능
- 모든 사망자 공개 정보를 종합해 남은 용의자 1~2명으로 압축

### 투표 전략
- 동률 발생 시 재투표에서 반드시 결론 → 무효표는 마피아에게만 유리
- 확신 없어도 더 의심스러운 쪽에 집중 → 틀려도 50% 확률

### 도망 선언 대응
```
마피아가 도망 선언하면:
1. 그 라운드 처형 투표 무효됨
2. 다음 밤 마피아 공격 발생 (한 명 사망)
3. 다음 라운드 낮에 반드시 처형
→ 도망 선언 = 마피아 확인 증거. 다음 라운드에 집중
```

### 경찰 최후 역할
- 아직 조사 결과 미공개라면 이번에 공개
- 마피아 확인 시 즉시 투표 집중 요청
- 경찰 본인도 공격받을 수 있으므로 정보를 미리 공개해두는 것이 안전
