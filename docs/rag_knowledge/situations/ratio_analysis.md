# 인원 비율별 전략

category: situation
phase: all
use_case: 현재 생존 인원 비율에 따른 전략 판단

---

## 인원 비율 계산법

```
위험도 판단:
마피아 수 / 전체 생존 수 = 마피아 비율

안전: 마피아 비율 < 30%
주의: 마피아 비율 30~40%
위험: 마피아 비율 40~50%
위기: 마피아 비율 ≥ 50% (마피아 승리 조건 달성)
```

---

## 상황별 전략

### 8명 생존 (마피아 2명 기준 — 25%)
```
시민팀: 안전한 편. 신중하게 정보 수집 후 확실한 목표 처형
마피아팀: 서두르지 말고 희생양 전술 지속. 시민팀이 실수하길 기다림
경찰: 조사 결과 1~2개 쌓일 때까지 신분 유지
의사: 경찰 보호 집중
```

### 6명 생존 (마피아 2명 — 33%)
```
시민팀: 주의 단계. 매 라운드 처형 필수. 실패하면 밤 사망으로 급격히 불리해짐
마피아팀: 이 타이밍에 경찰 제거 시도. 의사 보호를 피한 공격
경찰: 조사 결과 공개 적극 검토
의사: 경찰 신분 공개 후 즉시 경찰 보호
```

### 5명 생존 (마피아 2명 — 40%)
```
⚠️ 위험 단계
시민팀: 이번 라운드 처형 실패 시 시민 4 vs 마피아 2 → 밤 사망 후 3:2 = 위기
마피아팀: 한 라운드만 버티면 역전 가능. 최대한 시간 끌기
전략: 경찰 정보 즉시 공개, 강력한 투표 집중
```

### 4명 생존 (마피아 2명 — 50%)
```
🚨 즉시 마피아 승리 조건 달성
시민팀: 이미 패배 (마피아 수 ≥ 생존 시민 수)
→ 이 상황이 되지 않도록 4명 이상일 때 처리해야 함
```

### 5명 생존 (마피아 1명 — 20%)
```
시민팀: 유리. 용의자가 좁혀져 있으면 이번 라운드에 처리 가능
마피아팀: 도망 선언 준비, 밤 공격으로 2:1 역전 시도
→ 시민 4명 → 밤 1명 제거 → 시민 3 vs 마피아 1 → 한 라운드 더 버텨야
```

## 게임 종료 조건 계산표

| 생존 시민 | 마피아 수 | 상황 |
|-----------|-----------|------|
| 5 | 2 | 위험 — 이번 라운드 처형 필수 |
| 4 | 2 | 즉시 마피아 승리 |
| 3 | 1 | 위험 — 처형 필수 |
| 2 | 1 | 즉시 마피아 승리 |
| 4 | 1 | 시민 우세 — 신중히 처리 가능 |
