# 방어 발언 패턴

category: speech_pattern
speech_style: defense
use_case: 처형 위기에서 자신을 방어하거나 최후의 변론을 할 때

---

## 최후의 변론형

```
"마지막으로 한 말씀 드릴게요. 저는 진짜 시민입니다."
"제가 죽으면 우리 팀이 불리해져요. 신중하게 결정해주세요."
"처형하시기 전에, X를 한번 더 살펴봐주세요. 진짜 마피아가 저 뒤에 숨어있어요."
"틀리더라도 나중에 알게 될 거예요. 저는 억울합니다."
```

---

## 증거 제시형

```
"제가 시민이라는 증거는... 라운드 1부터 제 발언을 보세요. 일관성이 있어요."
"저는 X가 의심스럽다고 처음부터 말했어요. 마피아가 같은 팀을 의심할까요?"
"제 투표 기록 보세요. 항상 마피아 쪽에 투표했어요. 이게 증거 아닌가요?"
```

---

## 감정 호소형

```
"이렇게 억울할 수가 없어요. 저 정말 시민인데..."
"다들 왜 저를 믿어주지 않는 거예요. 계속 솔직하게 말했잖아요."
"마피아라면 이렇게 떨면서 말하겠어요? 저 진짜 무서워요."
```

---

## 전략적 자기 희생형 (진짜 시민이 팀을 위해 처형 수용)

```
"제가 틀렸다면 어쩔 수 없어요. 저를 처형하고 남은 분들이 끝내주세요."
"저보다 남은 팀원들이 더 중요해요. 잘 싸워주세요."
"제가 죽어도 X를 조심하세요. 그 사람이 진짜 마피아입니다."
```

---

## 마피아의 방어 발언 (연기)

```
"저 마피아 아니에요. 진짜예요. 경찰이 조사해줬으면 좋겠어요."
"저를 처형하면 진짜 마피아가 살아남아요. 잘 생각해보세요."
"제가 마피아라면 이렇게 많이 발언했을까요? 마피아는 숨어야 하잖아요."
"이건 함정이에요. 마피아가 저한테 몰아가고 있는 거예요."
```
