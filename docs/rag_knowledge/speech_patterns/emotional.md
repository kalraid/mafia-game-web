# 감정형 발언 패턴

category: speech_pattern
speech_style: emotional
use_case: 감정과 직관으로 의심하거나 호소할 때

---

## 직관 표현형

```
"논리적으로 설명하기 어렵지만, X를 보면 뭔가 느낌이 달라요."
"말로 설명할 순 없는데, X의 눈빛이 (발언 방식이) 계속 걸려요."
"제 직감이 틀린 적이 별로 없는데... X가 이상해요."
"분위기상 X가 마피아인 것 같아요. 여러분도 느끼지 않나요?"
```

---

## 호소형

```
"제발 이번엔 저 믿어주세요. X입니다."
"억울해서 말이 안 나와요. 제가 마피아면 지금 이러고 있겠어요?"
"우리 팀으로서 부탁드려요. X를 한번 더 의심해봐 주세요."
"저 혼자 이렇게 느끼는 건 아니겠죠? 다들 X가 이상하지 않나요?"
```

---

## 신뢰 표현형

```
"Y는 진짜 시민인 것 같아요. 말하는 방식이 달라요."
"Z랑은 처음부터 뭔가 통하는 느낌이었어요. 믿어도 될 것 같아요."
"W가 말할 때 진심이 느껴졌어요. 그 사람은 아닐 것 같아요."
```

---

## 분위기 전환형

```
"잠깐, 우리 너무 싸우는 거 아닌가요? 마피아가 우릴 이간질하는 거 아닐까요?"
"지금 분위기가 이상해요. 뭔가 유도당하는 느낌이에요."
"모두 진정하고 다시 처음부터 생각해봐요. 우리가 놓친 게 있는 것 같아요."
"감정적으로 흐르지 말고요. 지금 마피아가 원하는 게 이 혼란일 수 있어요."
```
