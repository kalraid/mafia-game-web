# 의심 돌리기(Deflection) 발언 패턴

category: speech_pattern
speech_style: deflection
use_case: 자신에게 쏠린 의심을 다른 사람에게 돌릴 때 (주로 마피아 사용)

---

## 즉각 반격형

```
"저를 의심하시는 건 알겠는데, 왜 X는 아무도 안 보는 거예요?"
"저한테 화살 돌리기 전에, 정작 아무 발언 없이 조용히 있는 Y는요?"
"잠깐, 저 지목하는 분이 오히려 더 수상한데요?"
```

---

## 비교 전환형

```
"저보다 Z의 행동이 훨씬 더 의심스럽지 않아요? 투표 패턴 보셨어요?"
"저를 의심할 시간에 W를 한번 잘 보세요. 처음부터 말이 없었잖아요."
"저는 처음부터 제 의견을 분명히 밝혔어요. 반면 X는 항상 다수 의견만 따라가잖아요."
```

---

## 논리 반박형

```
"제가 마피아라면 이렇게 발언 많이 했을까요? 오히려 조용히 있는 게 낫잖아요."
"저를 마피아로 보는 근거가 뭔가요? 그 근거라면 Y도 마피아일 수 있어요."
"저한테 의심의 눈초리를 보내는 것 자체가 마피아의 전술일 수 있어요."
```

---

## 연대 호소형

```
"우리가 서로 싸우면 마피아만 이득이에요. 진짜 위협에 집중합시다."
"저를 의심하는 대신 같이 X를 살펴봐요. 훨씬 더 수상해요."
"이 혼란이 마피아가 원하는 거예요. 우리 냉정하게 봐야 해요."
```

---

## 자기 변호형

```
"저는 지금까지 제 추리를 투명하게 공개했어요. 숨긴 게 없어요."
"제가 말한 것들 다시 돌아보세요. 어디가 이상한가요?"
"저를 마피아로 처리하면 나중에 후회할 거예요."
"경찰이 있다면 저를 조사해보세요. 확인해드릴게요."
```
