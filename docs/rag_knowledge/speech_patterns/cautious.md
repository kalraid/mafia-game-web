# 조심스러운 발언 패턴

category: speech_pattern
speech_style: cautious
use_case: 확신이 없거나 신중하게 의심을 표현할 때

---

## 관찰 표현형

```
"확실하지는 않은데, X의 발언이 조금 신경 쓰이네요."
"제가 틀릴 수 있지만, X가 처음부터 조용한 게 좀 이상하게 느껴져요."
"아직 판단하기 이른 것 같아요. 조금 더 지켜볼게요."
"저도 X를 의심하는 건 아닌데, 발언 패턴이 좀 다른 것 같아서요."
```

---

## 의견 유보형

```
"음... 저는 아직 잘 모르겠어요. 모두 어떻게 생각하세요?"
"지금 상황이 복잡해서 섣불리 판단하기가 어렵네요."
"저는 이번 라운드 기권할까 생각 중이에요. 정보가 너무 부족해서요."
"조금 더 발언을 들어보고 결정하고 싶어요."
```

---

## 질문형 (간접 의심)

```
"X에게 하나만 물어봐도 될까요? 라운드 1에 왜 그 사람에게 투표하셨어요?"
"혹시 누가 X의 발언 패턴을 정리해줄 수 있나요? 제가 놓친 게 있는 것 같아서요."
"Y랑 Z 중에 어떻게 생각하세요? 둘 다 이상한 것 같기도 하고..."
```

---

## 중립적 의견 제시형

```
"A도 의심스럽지만 B도 완전히 믿기는 어려워요."
"저는 아직 모두에게 가능성이 있다고 봐요."
"지금 우리가 너무 서두르는 건 아닐까요? 마피아가 원하는 게 혼란일 수도 있어요."
"틀리더라도 괜찮으니까 모두 솔직하게 생각을 나눠봐요."
```
