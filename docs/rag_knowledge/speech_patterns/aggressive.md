# 공격적 발언 패턴

category: speech_pattern
speech_style: aggressive
use_case: 강하게 특정인을 지목하거나 의심할 때

---

## 직접 지목형

```
"저는 확실히 X가 마피아라고 봅니다. 지금까지 발언을 보세요."
"X, 지금 당신 말이 앞뒤가 안 맞아요. 설명해보세요."
"더 이상 X를 봐줄 이유가 없어요. 이번 라운드에 처리해야 합니다."
"X의 투표 패턴이 너무 수상해요. 항상 마피아에게 유리한 방향으로만 투표하잖아요."
"저는 X에게 한 표 던집니다. 이유는 명확합니다."
```

---

## 역공격형 (자신이 의심받을 때)

```
"저를 의심하는 분들한테 묻고 싶은데요, 근거가 뭔가요?"
"저 마피아면 이렇게 적극적으로 발언했을 것 같아요? 마피아는 조용히 있는 게 유리하잖아요."
"저를 지목하기 전에 X부터 설명해보세요. 왜 X는 아무도 안 의심하는 거죠?"
"잠깐요, 지금 저한테 화살 돌리는 거 마피아 전술 아닌가요?"
```

---

## 주장 강화형

```
"제 말을 들어보세요. 라운드 1부터 지금까지 X의 행동을 정리해보면요..."
"시간이 없어요. 지금 결정해야 합니다. X입니다."
"다른 의견도 존중하지만, 저는 확신합니다. X를 처리하지 않으면 우리가 집니다."
"경찰 결과까지 나왔잖아요. 더 이상 머뭇거릴 필요가 없어요."
```

---

## 투표 독려형

```
"모두 X에게 투표해주세요. 이번이 기회입니다."
"지금 흔들리지 마세요. X입니다."
"마지막 변론 듣고 판단하세요. 하지만 저는 바꾸지 않을 겁니다."
```
