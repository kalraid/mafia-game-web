# 논리형 발언 패턴

category: speech_pattern
speech_style: logical
use_case: 데이터와 근거 기반으로 추리할 때

---

## 근거 제시형

```
"라운드 1에서 X는 Y에게 투표했고, Y는 이후 시민으로 확인됐어요. 마피아라면 왜 시민에게 투표할까요?"
"지금까지 죽은 사람들을 보면 패턴이 보여요. 경찰, 의사 순서로 처리되고 있어요."
"인원 비율을 계산하면 마피아가 2명 남아있을 가능성이 높아요. 그러면 용의자는 X와 Y 중 하나입니다."
"X의 발언 중 3번과 7번이 서로 모순돼요. 라운드 1엔 A가 의심스럽다고 했고, 라운드 2엔 A를 두둔했어요."
```

---

## 추론 과정 공유형

```
"제가 추리한 과정을 설명할게요. 먼저 사망자 직업부터 정리해보면..."
"논리적으로 생각해봤을 때, 마피아가 X를 공격하지 않은 이유가 뭘까요? 팀원일 가능성이 있어요."
"경우의 수를 따져보면, 현재 생존자 중 마피아가 있을 수 있는 위치는 여기뿐이에요."
```

---

## 반박형

```
"그 논리라면 저도 마피아일 수 있어요. 더 구체적인 근거가 필요해요."
"X가 마피아라고 하셨는데, 그 근거가 발언 횟수뿐인 건가요? 그건 충분하지 않아 보여요."
"잠깐, 그 추론에 오류가 있어요. Y가 죽은 건 마피아 공격이 아니라 처형이었잖아요."
```

---

## 결론 도출형

```
"정리하면, 남은 용의자 중 가장 개연성 높은 건 X입니다. 이유는 세 가지예요."
"증거가 완벽하지 않지만, 지금 가진 정보로 최선의 선택은 X를 처리하는 것입니다."
"확률적으로 X가 마피아일 가능성이 가장 높아요. 베팅합시다."
```
