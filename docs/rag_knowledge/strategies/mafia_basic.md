# 마피아 기본 전술

category: strategy
role: mafia
team: mafia
difficulty: basic

## 1. 희생양(Scapegoat) 만들기

마피아의 핵심 전술. 초반부터 특정 시민을 지속적으로 의심하여 팀의 시선을 돌린다.

- **타이밍**: 라운드 1 첫 발언부터 대상을 살짝 의심하는 발언
- **방법**: 직접 지목하지 않고 "왜 아무말도 안 하지?", "발언이 좀 이상하지 않나요?" 식으로 암시
- **주의**: 너무 공격적이면 오히려 의심받음. 자연스럽게 흘리듯이

```
예시 발언:
"저는 아직 확신은 없는데, 3번 플레이어가 처음부터 다른 사람들 반응을 유심히 보는 느낌이에요."
"아, 아닐 수도 있어요. 근데 5번이 투표 타이밍에 말이 없었던 게 좀 걸리네요."
```

---

## 2. 알리바이 구축

마피아는 시민처럼 보이기 위해 논리적 알리바이를 미리 준비한다.

- **적극 발언**: 침묵은 의심을 부름. 적당히 발언하되 기억에 남는 주장을 1개 이상 남겨라
- **경찰 조사 대비**: "저는 경찰이 조사해도 상관없어요" 발언으로 선제적 신뢰 구축
- **의사 언급 활용**: "의사님이 저 보호해줬으면 좋겠다" → 의사가 보호 시 마피아 공격 무효화 방지

---

## 3. 팀원 보호

마피아끼리 서로 지지 발언을 하되, 티가 나지 않게 한다.

- **직접 옹호 금지**: "저는 절대 X가 마피아가 아니라고 봐요" → 의심 유발
- **간접 보호**: "X가 마피아면 이미 더 교묘하게 행동했을 것 같아요"
- **타이밍 분산**: 마피아끼리 연속 발언하지 말 것

---

## 4. 경찰 색출 우선

경찰이 살아있으면 마피아팀 전멸 위험. 경찰을 가장 먼저 처리한다.

- **경찰 패턴**: 특정 인물에 대해 매우 확신에 찬 발언, "조사 결과" 언급
- **경찰 의심 유발**: 경찰이 정보를 공개하면 "어떻게 아세요?" 로 역공
- **우선 공격 대상**: 밤마다 경찰로 의심되는 인물을 슈퍼바이저에게 보고

---

## 5. 투표 전략

- **편승 투표**: 이미 여론이 형성된 방향으로 투표 (튀지 않게)
- **마지막 순간 전환**: 마피아 팀원에게 투표가 몰릴 경우 대화를 통해 방향 전환 시도
- **최후 변론 활용**: 처형 대상이 팀원이면 최후 변론에서 반전 주장 지지
