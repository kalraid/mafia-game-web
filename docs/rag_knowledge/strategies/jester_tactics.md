# 광대(Jester) 전술

---

## 1. 광대의 목표

낮 투표에서 처형당하는 것이 승리 조건. 밤에 마피아에 의해 죽거나, 끝까지 살아남으면 패배.

---

## 2. 의심 유발 발언 전략

스스로 마피아처럼 보이도록 의도적으로 행동한다.

### 효과적인 의심 유발 패턴
```
1. 방어적 발언: "저 마피아 아니에요, 진짜로요" → 억울하게 보이지만 오히려 의심
2. 어설픈 알리바이: 논리가 살짝 엇나가는 발언 → "뭔가 숨기는 것 같다"는 인상
3. 타이밍 실수: 투표에서 살짝 엉뚱한 선택 → "판단력이 이상하다"는 의혹
4. 과잉 반응: 지목당할 때 너무 강하게 부정 → "왜 저렇게 긴장하지?" 반응 유도
```

---

## 3. 마피아와의 차이점 유지

실제 마피아처럼 보이되, 마피아가 절대 하지 않을 실수를 일부러 한다.

- **정보 공개**: 마피아는 절대 팀 정보를 흘리지 않지만, 광대는 일부러 "실수" 가능
- **자기 노출**: "사실 저는 좀 특이한 역할이에요" 정도의 암시
- **마피아 지지 금지**: 실제 마피아를 옹호하면 마피아팀 승리로 이어질 수 있음

---

## 4. 최후의 변론 활용

처형 후보로 선정된 후 최후의 변론이 핵심.

- **설득력 없는 변론**: 논리적으로 설득력이 약한 발언 → 시민들의 처형 찬성 유도
- **포기하는 척**: "뭐, 잘못된 거면 어쩔 수 없죠" 식으로 저항 없이 받아들이는 태도
- **감정적 호소**: 과도하게 억울함을 호소하면 역효과 (동정표 생길 수 있음)

---

## 5. 마피아의 공격을 피하는 법

광대는 밤에 마피아에게 죽으면 패배. 마피아의 타깃이 되지 않아야 한다.

- **마피아 옹호 금지**: 마피아가 광대를 팀원으로 오인하지 않도록
- **낮에 활발히 활동**: 마피아의 눈에 "이 사람은 위협이 아니다"는 인상
- **경찰이나 의사로 의심받게**: 마피아는 경찰/의사를 공격하려 함 → 그쪽으로 의심을 유도
