# 게임 룰 요약 (Agent 조회용)

category: rules
use_case: Agent가 게임 중 룰을 빠르게 확인할 때

---

## 기본 진행

```
[낮] 전체 채팅(3분) → 투표(1분) → 최후의 변론(30초) → 찬반투표(30초) → 처형/무효
[밤] 마피아 협의(1분30초) → 능력 사용(1분) → 밤 결과 처리
→ 승리 조건 판정 → 반복
```

---

## 직업별 핵심 정보

| 직업 | 진영 | 능력 | 승리 조건 |
|------|------|------|-----------|
| 시민 | 시민 | 없음 | 마피아 전원 제거 |
| 경찰 | 시민 | 밤: 1명 진영 확인 | 마피아 전원 제거 |
| 의사 | 시민 | 밤: 1명 보호 | 마피아 전원 제거 |
| 점쟁이 | 시민 | 밤: 1명 직업군 확인 | 마피아 전원 제거 |
| 마피아 | 마피아 | 밤: 1명 제거 | 마피아 수 ≥ 시민 수 |
| 킬러 | 마피아 | 밤: 보호 무시 공격(1회) | 마피아 수 ≥ 시민 수 |
| 광대 | 중립 | 없음 | 낮 투표로 처형당함 |
| 스파이 | 중립 | 밤: 마피아 채팅 열람 | 게임 종료 시 생존 |

---

## 핵심 제한 사항

- **경찰**: 동일 대상 연속 2회 조사 불가
- **의사**: 동일 대상 연속 2회 보호 불가 / 자기 보호 게임당 1회
- **킬러**: 관통 공격 게임당 1회
- **마피아 도망**: 게임당 1회, 처형 투표 무효화

---

## 투표 규칙

- 동률 → 재투표 1회
- 재투표 동률 → 무투표 (처형 없음)
- 찬반 투표 동률 → 처형 확정
- AI는 반드시 투표 (기권 시 랜덤)

---

## 특수 이벤트

- **첫날 밤**: 마피아 공격 없음 (탐색 밤)
- **광대 처형**: 즉시 게임 종료 (광대 단독 승리)
- **침묵 AI**: 낮 3분간 발언 없으면 신뢰도 -1 패널티

---

## 정보 공개 규칙

- 사망자 직업: **전체 공개** (기본)
- 경찰 조사 결과: **경찰 본인만**
- 의사 보호 성공 여부: **의사 본인만**
- 스파이 마피아 채팅: **스파이만**
