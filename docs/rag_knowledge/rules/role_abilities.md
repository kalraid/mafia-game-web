# 직업별 능력 상세

category: rules
use_case: Agent가 자신의 능력과 제한을 확인할 때

---

## 경찰 (Detective)

### 능력
- **조사**: 밤 Phase에 생존 플레이어 1명의 진영(마피아 여부) 확인
- 결과: "마피아입니다" 또는 "시민입니다" (마피아/킬러는 마피아로 표시)

### 제한
- 동일 대상 연속 2회 조사 불가
- 조사 결과는 경찰 본인에게만 전달
- 점쟁이보다 정확 (직업군이 아닌 정확한 진영 확인)

### 행동 가이드
```
능력 사용 API: POST /game/{id}/ability
Body: { "player_name": "경찰이름", "ability": "investigate", "target": "조사대상이름" }
```

## 의사 (Doctor)

### 능력
- **보호**: 밤 Phase에 생존 플레이어 1명을 보호 → 마피아 공격 무효화
- 킬러의 관통 공격은 보호 불가 (게임당 1회 한정)

### 제한
- 동일 대상 연속 2회 보호 불가
- 자기 자신 보호: 게임당 1회만 허용
- 보호 성공/실패 결과는 의사 본인에게만 전달

### 행동 가이드
```
능력 사용 API: POST /game/{id}/ability
Body: { "player_name": "의사이름", "ability": "protect", "target": "보호대상이름" }
※ "protect"는 서버에서 "heal"로 자동 매핑됨
```

## 점쟁이 (Fortune Teller)

### 능력
- **점괘**: 밤 Phase에 생존 플레이어 1명의 직업군 확인
- 결과: "시민계열", "마피아계열", "중립" 중 하나

### 특징
- 경찰보다 범위가 넓어 덜 정확 (킬러도 "마피아계열", 광대/스파이는 "중립")
- 12인 이상 게임에서만 등장

---

## 마피아 (Mafia)

### 능력
- **공격**: 밤 Phase에 생존 플레이어 1명 지목 → 팀 협의 후 최종 1명 처치
- 마피아 여러 명이면 각자 지목 → 슈퍼바이저가 최종 결정

### 채팅 채널
- 밤 Phase에 `mafia_secret` 채널에서 팀원끼리 협의 가능
- 스파이가 엿볼 수 있으니 직접적인 신분 언급 주의

---

## 킬러 (Killer)

### 능력
- **관통 공격**: 의사 보호를 무시하고 처치 (게임당 1회)
- 일반 마피아 공격과 별도로 사용 가능

### 사용 전략
- 의사가 특정인을 보호하는 것이 확실한 날 사용
- 경찰을 관통 공격으로 제거하는 것이 최고 효율

---

## 광대 (Jester)

### 능력
- 없음. 특수 승리 조건만 존재

### 승리 조건
- 낮 투표(찬반 투표 포함)에서 처형당하면 즉시 단독 승리
- 마피아에게 밤에 죽거나, 게임 종료까지 생존하면 패배

---

## 스파이 (Spy)

### 능력
- **엿듣기**: 밤 Phase에 마피아 채팅 채널(`mafia_secret`) 읽기 가능
- 읽기 전용 (마피아 채팅에 메시지 전송 불가)

### 승리 조건
- 게임 종료 시까지 생존하면 승리 (진영 무관)
- 15인 이상 게임에서만 등장
